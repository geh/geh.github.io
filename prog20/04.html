Los ejercicios se deben hacer dentro de JSLinux, y con el editor de texto vi.

1. Variables de tipo float

Se puede estimar la edad de una persona en función de su número de DNI argentino?

Algunos se arriesgaron a proponer la fórmula siguiente:

(-0.000001508 * dni) + 77.779

  1. Implementar un programa que pide su DNI al usuario y estime su edad, usando dos variables: int dni y float edad.

  2. Modificar el programa de la forma siguiente: si el dni es mayor a 90 millones, o es de 18 o 19 millones, no se estima la edad.

2. Bucle while y condiciones

Hacer un programa selectivo.c que imprima las secuencias siguientes:

  1. los enteros de 0 a 100 salvo los que son múltiples de 7.
  2. los enteros de 0 a 30 salvo los que son múltiples de 3.
  3. los enteros de 50 a 500 salvo los que son múltiples de 11

Recordar que n es múltiple de m siempre y cuando la condición n % m == 0 es verdadera.

Tareas para el 27/4

Tarea 1: Calcular sumas y productos de series

Hacer los ejercicios siguientes en jslinux:

  • escribir un programa suma1.c que pida el valor entero m al usuario, y luego calcule e imprime la suma 1 + 2 + ... + m.
  • escribir un programa suma2.c que pida los valores enteros n y m al usuario, y luego calcule e imprime la suma de todos los enteros entre 1 y n que no son divisibles por m. (Comprobar: si n=100 y m=2, la suma vale 2500).
  • escribir un programa prod1.c que pida el valor entero n al usuario, y luego calcule e imprime el producto 1 * 2 * ... * n. (Comprobar: si n=5, el resultado es 120).
  • escribir un programa prod2.c que pida el valor entero n al usuario, y luego calcule e imprime el producto n * (n + 1) * ... * (n + 5). (Comprobar: si n=5, el resultado vale 5 * 6 * 7 * 8 * 9 * 10 = 151200).

Tarea 2: Collatz

Escribir un programa que pide un entero n al usuario e imprima la secuencia de Collatz desde ese entero n hasta 1, y luego imprima cuántos pasos tiene la sucesión.

Consultar https://es.wikipedia.org/wiki/Conjetura_de_Collatz y usar los ejemplos 6, 11 y 27 para comprobar su programa.

Tarea 3: Pruebas de escritorio

Los programas siguientes son presentados sin el texto común main(){...}.

Para cada uno de esos programas, armar una tabla donde cada línea corresponde a la sentencia ejecutada, y cada columna a cada variable que aparece en el programa. Además, agregar una columna cuyo título sea “condición”.

En el caso de condiciones (de if o while) escribir la expresión de la condición.

Programas con condiciones if, if-else

int a = 1;
if (a > 10){
    a = a - 15;
} else {
    a = a + 10;
} 
int a = 10;
int b = 1;
int c = 0;
if (a <= 10 && b >= 1){
    c = c + 1;
}
if (a < 10){
    c = c + 1;
}
if (b > 1){
    c = c + 1;
} 
int a = 99;
int c = 0;
if (a >= 80){
    c = c + 1;
} else if (a >= 90) {
    c = c + 1;
} else if (a >= 95) {
    c = c + 1;
}
int b = 9;
int c = 0;
if (b > 10){
    c = 1;
} else if (b > 5){
    c = 2;
} else {
    c = 3;
}

Programas con bucles while

int i = 0;
int a = 0;
while (i < 5){
    i = i + 2;
}
int i = 0;
int a = 0;
while (i < 4){
    if (i % 2 == 0)
        a = a + 1;
    else
        a = a - 1;
    i = i + 1;
}
int i = 5;
int a = 0;

    else
