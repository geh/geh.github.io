1. Conocer las cadena de caracteres terminadas en cero

En un programa en PythonTutor declará las dos variables siguientes:

  char * holaPuntero = "Hola!";
  char holaArreglo[5] = "Hola!";

  return 0;

Ejecuta este programa. Visualizá en la diferencia entre esas dos variables en memoria.

Luego, con un bucle while, recorré la cadena holaPuntero hasta encontrar el valor 0.

Para cada iteración del bucle, imprimí el carácter.

Una vez que esto funciona, cambiá el valor de la cadena holaPuntero a una cadena más larga como "Bienvenidos!" y volvé a visualizar la ejecución para confirmar que tu código funciona.

2. Modificar una string dada en línea de comando

Como referencia, vas a necesitar la Sección 7.5 del manual “Aprenda C”.

Escribí un programa cuyo ejecutable pueda ser usado de la forma siguiente:

$ ./a.out hola
ubyn
$ ./a.out ubyn
hola
$ ./a.out universidad
havirefvqnq
$ ./a.out havirefvqnq
universidad

Esta transformación se llama rot13, para “rotación de 13 caracteres”. Cada letra se cambia por la letra ubicada 13 pasos adelante en el alfabeto. Si llegamos al final del alfabeto, se vuelve al principio. No se toma en cuenta la letra ñ.

En este ejercicio, podés suponer que el usuario solo te proveé una palabra escrita en minúsculas. Pero si lo resolvés temprano, podés extenderlo para que tome cualquier palabra, y que solo aplique rot13 sobre los caracteres alfabéticos (minúsculas y majúsculas).

4. Encontrar el valor mediano

Reutilizá el padrón de programa del ejercicio anterior en un programa nuevo mediano.c.

Agregá a este programa el código necesario para calcular e imprimir el valor mediano de los valores ingresados, suponiendi que siempre se ingresa una cantidad impar de valores que no hay valores repetidos.

El valor mediano es el valor ingresado m tal que, entre los demás valores, la mitad es mayor a m y la otra es menor a m.

Una forma de resolver esto es ordenar el arreglo a, pero no sabemos hacerlo por el momento así que vamos a hacerlo de forma más simple (y menos eficiente):

  • para cada valor x del arreglo a, hacer lo siguiente:
    • recorrer los otros valores del arreglo a, calcular la cantidad c1 de elementos mayores a x, y la cantidad c2 de valores menores a x.
    • si c1 es igual c2, interrumpir el proceso y publicar x como valor mediano.

Ejemplos de ejecución:

tcc -run mediano.c 4 5 1 2 3
El valor mediano es: 3.


tcc -run mediano.c 4 5 1 20 3 0 30
El valor mediano es: 4.

5. Ordenar un arreglo de valores: el método del duende

Reutilizá el padrón de programa del ejercicio anterior en un programa nuevo duende.c.

Vas a implementar un algoritmo que ordena estos valores, el método del duende.

La idea de este algoritmo es imaginarse un duende del jardín ordenando macetas. La única acción que puede hacer el duende (aparte moverse) es intercambiar dos macetas una al lado de la otra.

En este programa el duende es representado por la variable de conteo i.

El duende empieza por el principio del arreglo (posición 0) y termina cuando llega al final (posición argc-2):

  • si está mirando al primer elemento de la lista (a[0]), o si el elemento que mira es mayor o igual al elemento que está justo a la izquierda (a[i] >= a[i-1]), entonces hace un paso a la derecha sin intercambiar nada.
  • si el elemento que está mirando es menor al que está justo a la izquierda (a[i] < a[i-1]), entonces intercambia los dos valores, y hace un paso a la izquierda.

El duende repite este proceso hasta que llegue al final del arreglo.

Dado que esto no es un bucle “clasico” donde la variable de conteo se incrementa de forma regular, es recomendable utilizar un bucle while y no un for.
