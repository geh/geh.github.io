# 1. Impresión de mensajes (2 puntos)

 1. Realizar un programa que imprime el mensaje siguiente (respetando los saltos de línea):

    ~~~
    al sur al sur
    esta quieta esperando
    Montevideo 
    ~~~

 2. Modificar el programa para que imprima además, en otra línea:

    ~~~
    "Mario Benedetti"
    ~~~ 

# 2. Variables carácter y cadena de caracteres (2 punto)

 1. Realizar un programa que:
    * declare una variable carácter con algun nombre
    * pida un carácter al usuario con `getchar` y lo cargue en esa variable
    * muestre el mensaje:

    ~~~
    El caracter ingresado es:
    ~~~
    seguido del carácter, en una misma línea.

 2. Modificar el programa agregándole lo siguiente:
    * declarar una variable cadena de caracteres con algun nombre y con valor `"Me gusta repetir!\n"`.
    * solo usando esta variable, hacer que el programa muestre:

    ~~~
    Me gusta repetir!
    Me gusta repetir!
    Me gusta repetir!
    Me gusta repetir!
    ~~~

# 3. Estructuras condicionales (2 puntos)

 1. Realizar un programa que:
    * declare una variable carácter con algun nombre
    * pida un carácter al usuario con `getchar` y lo cargue en esa variable
    * si el carácter ingresado es igual a `'z'`, imprimir el mensaje `"Buenas noches!"`.
 2. Modificar el programa de la forma siguiente:
    * si el valor numérico del carácter ingresado es menor o igual a 64, mostrar:
      `"Este caracter es raro, su valor numerico es:"` seguido del carácter mostrado como entero,
    * sino mostrar `"Este caracter me gusta:"` seguido del carácter mostrado como carácter.

# 4. Lectura de enteros, expresiones lógicas (2 puntos)

En este ejercicio, no olvidarse de hacer las declaraciones de variables necesarias
al principio del cuerpo de la función `main`.

  1. Realizar un programa que lea por teclado un número entero.
     Si es mayor o igual a 1000, o menor o igual a 0, imprimir `"Fuera de rango."`,
     sino imprimir `"OK."`
  2. Luego, pedir un segundo número entero al usuario.
     Si ese segundo número es igual al primero, imprimir `"Es igual."`, sino imprimir `"Es distinto."`.
  3. Luego, pedir un tercer número entero al usuario.
     Si los tres valores ingresados son menores a 10, imprimir `"Todos los numeros son menores a diez"`.

# 5. Problemas (2 puntos)

Contestar cada problema en un programa distinto.

 1. Realizar un programa que pida cargar una fecha cualquiera (primero el día, luego el mes),
    luego verificar si dicha fecha corresponde a Navidad.
 2. Realizar un programa que pida cargar tres números distintos.
    Imprimir el mayor de ellos.
