Parte 3: sentencias condicionales

3.1. Comparaciones

En un programa nuevo comparar.c:

  • declarar dos enteros a y b
  • pedir dos valores al usuario para cargar en esas variables a y b
  • si a > b, mostrar “a es mayor” (tal cual, sin reemplazo de valores)
  • sino, mostrar “b es mayor”
  • finalmente, mostrar el valor absoluto de la diferencia entre a y b

Si x es una expresión aritmetica, abs(x) nos da su valor absoluto.

3.2. getchar, comparaciones y estructuras condicionales

  1. Escribir un progama que pida al usuario un carácter, luego si el carácter es 'a' imprime “Hola!”, sino imprime “Buen dia.”, y luego imprime (en todos casos) “Adios!”.

  2. Escribir un programa que imprima primero “Ingrese una letra, mayuscula o minuscula”. Pedir un carácter con getchar. Luego, usando una sentencia if con un solo operador relacional, si el carácter es una mayúscula imprimir “Es una mayuscula!”, sino es imprimir “Es una minúscula!”. (Estamos suponiendo que el usuario ingresa o una mayúscula o una minúscula, ninguna otra cosa).

    Ayuda: los caracteres están ordenados según algo que se llama “código ASCII”. Las siguientes comparaciones son verdaderas: 'a' < 'b' , 'b' < 'm', 'X' > 'D'. Por convención, n < m siempre vale cuando n tiene el valor de una mayúscula y m de una minúscula (o sea, las mayúsculas son menores a las minúsculas).
